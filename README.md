# Booking Web App
a booking web app using graphql technology
- Frontend: (localhost:3000)
  - React
  - Apollo Client
- Backend: (localhost:1234/graphql)
  - Express
  - Express-graphql
- Database:
  - MongoDB

## Requirements

Node >= 10.0

npm >= 6.4

docker >= 1.8.0

## Inital setup

docker-compose build

## Run

docker-compose up

## Stop all services

docker-compose down
